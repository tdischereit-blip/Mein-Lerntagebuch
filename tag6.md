# Tag 6 – [07.06.2026]

## Was habe ich heute gelernt?
GitHub Pages macht aus einem Repo eine echte Website.
Die README.md wird automatisch zur Startseite.
Jeder Push auf main aktualisiert die Seite.

## Welches Problem bin ich begegnet?
Die Seite hat nach dem Aktivieren eine 404-Fehlerseite
gezeigt statt der README.

## Wie habe ich es gelöst?
In Settings → Pages den Branch auf "main" und
den Ordner auf "/ (root)" gesetzt. Nach ~2 Minuten
war die Seite online.
